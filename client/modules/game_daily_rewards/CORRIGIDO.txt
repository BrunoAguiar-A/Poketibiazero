   - Evita erro de módulo não carregado

   - Verifica se game_interface existe antes de usar
   - Verifica se client_topmenu existe antes de adicionar botão

3. ✅ Adicionada proteção em todas as funções
   - show() verifica se mainWindow existe
   - hide() verifica se mainWindow existe
   - toggleWindow() verifica se mainWindow existe
   - closeWindow() verifica se mainWindow existe

4. ✅ Adicionada flag de inicialização
   - Evita inicialização dupla
   - Garante que init() é chamado apenas uma vez

COMO TESTAR AGORA

1. REINICIE O CLIENTE COMPLETAMENTE
   - Feche o cliente
   - Aguarde 5 segundos
   - Abra o cliente novamente

2. PROCURE PELA MENSAGEM NO CONSOLE
   - Abra o console (Ctrl+Shift+C ou F12)
   - Procure por: ">> Módulo de Daily Rewards carregado com sucesso!"
   - Se vir esta mensagem, o módulo foi carregado corretamente

3. PROCURE PELO BOTÃO NO TOPMENU
   - Olhe para o topo da tela
   - Procure por um ícone de presente (gift icon)
   - Clique no ícone para abrir a janela

4. SE O BOTÃO NÃO APARECER
   - Abra o console
   - Execute: modules.game_daily_rewards.show()
   - A janela deve abrir
