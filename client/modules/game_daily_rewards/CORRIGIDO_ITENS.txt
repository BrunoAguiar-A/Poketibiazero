================================================================================
COMO EDITAR OS ITENS
================================================================================

Se quiser mudar os itens:

1. Abra: game_daily_rewards.lua
2. Localize: local rewardsData = {
3. Altere os valores de clientId, count e name
4. Salve (Ctrl+S)
5. Reinicie o cliente

EXEMPLO:
  ANTES: [1] = {clientId = 26731, count = 1, name = "Leaf Stone"},
  DEPOIS: [1] = {clientId = 12345, count = 2, name = "Meu Item"},

================================================================================
ESTRUTURA DO ITEM
================================================================================

[DIA] = {clientId = ID, count = QUANTIDADE, name = "NOME"}

Onde:
  • DIA: Número do dia (1-21)
  • clientId: ID do item no cliente
  • count: Quantidade do item
  • name: Nome descritivo (apenas para referência)

================================================================================
NOTAS IMPORTANTES
================================================================================

1. Os clientIds devem ser válidos no cliente
2. A quantidade (count) deve ser um número inteiro positivo
3. O nome (name) é apenas para referência, não afeta o jogo
4. Máximo de 21 dias (não altere o número de itens)
5. Após editar, reinicie o cliente para aplicar as mudanças
6. Os itens agora aparecem corretamente na interface!
